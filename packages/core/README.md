# @mcify/core

Builder library for mcify MCP servers. Provides typed primitives to define tools, resources, prompts, and auth, plus shared schema and logging helpers.

> **Status:** scaffold. Public API arrives in Phase A.3 (see [PHASES.md](https://github.com/Lelemon-studio/mcify/blob/main/../specs/) in the workspace planning).

## Install

```bash
npm install @mcify/core
```

## Usage

API will land in `0.0.1-alpha.x`. Subscribe to the repo for updates.
